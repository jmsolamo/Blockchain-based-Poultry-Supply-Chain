Blockchain-based Poultry Supply Chain

Poultry Farming and Distribution
  -Tracks chicken or egg products from farmers to distributors/retailers.
  -Ensures transparency, traceability, and accountability in the supply chain.

The system is a smart contract on Ethereum that allows authorized farmers to register poultry products and transfer ownership through the supply chain. It maintains:
Product information: name, quantity, origin, date created.
Ownership tracking: history of who owned the product at each stage.
Status monitoring: product status updates from Created → InTransit → Delivered.
Role-based access control: only admin can add farmers/distributors; only authorized users can register or transfer products.

Key benefits:
Real-time tracking of products.
Immutable record of ownership.
Enhanced transparency for end consumers and regulators.

Contract Features
Feature	Description
Admin Role	Admin can authorize farmers and distributors.
Farmer Registration	Authorized farmers can register new products.
Ownership Transfer	Farmers can transfer products to distributors; distributors can mark as delivered.
Status Tracking	Tracks product through stages: Created, InTransit, Delivered.
Ownership History	Maintains a complete record of all owners for each product.
